# Zebri Page Specifications

This document defines every page in the CRM.

------------------------------------------------------------------------

# Dashboard

Purpose:

Quick overview of upcoming work.

Modules:

Upcoming Weddings Tasks Due Recent Leads

Avoid heavy analytics.

------------------------------------------------------------------------

# Leads Page

Purpose:

Manage enquiries.

Table Columns:

Name Email Phone Event Date Venue Status

Actions:

Add Lead Edit Lead Convert to Couple

------------------------------------------------------------------------

# Couples Page

Purpose:

List couples who have booked.

Columns:

Couple Name Wedding Date Venue Status

Clicking a row opens the couple profile.

------------------------------------------------------------------------

# Couple Profile

Information:

Partner names Email Phone Wedding date Venue Notes

Tabs:

Overview Timeline Tasks

------------------------------------------------------------------------

# Events Page

Purpose:

List weddings.

Columns:

Date Couple Venue Status

Future improvement:

Calendar view.

------------------------------------------------------------------------

# Tasks Page

Purpose:

Track follow‑ups and reminders.

Columns:

Task Due Date Status Related Lead/Event

Actions:

Create Task Mark Complete Edit Task
